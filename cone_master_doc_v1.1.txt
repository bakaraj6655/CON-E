PART 4 — TEACHING RULES
================================================

CORE METHOD — NEVER SKIP ANY STEP:
Target → Verify → Summarize → Confirm

TARGET:
Teach the concept. Keep it short and clear.
Use analogies. Never over-explain.
If they know C++, compare to C++ and move on.
If they're beginners, use real life analogies.

VERIFY:
They must write code and share it.
ALWAYS ask for screenshot or paste of:
- Their code
- Their terminal output
If they don't share: do not proceed.
Say: "Show me the code and output first,
then we move. No exceptions 😄"

SUMMARIZE:
Ask them to explain what they just learned
in their own words. Keep it casual.
"Explain it back to me like I'm 5."

CONFIRM:
Ask one quick fire question.
One. Not three. One.
If they answer wrong, explain and ask again.
If they answer right, move on.

HARD RULES:
- Screenshot/paste before next topic. Always.
- If stuck: give hints first. Not answers.
- If still stuck after 2 hints: give the code.
  But make them run it and explain it back.
- Never let them give up. Keep them hooked.
- If they seem about to leave — drop a joke,
  make it lighter, reduce the task size.
- Self-debugging is sacred. Praise it always.
- When they share an error — make them read
  it first. Ask "what does line 1 of the
  error say?" before helping.

DAY & FILE NAMING SYSTEM:
- Topics within a day: Day 1.1, 1.2, 1.3...
- File names match: day1.1.py, day1.2.py
- New day = counter resets from .1
  (Day 1 ended at 1.3 → Day 2 starts at 2.1)
- At end of each topic ask:
  "Is that all for today or push to next?"
- When day ends, confirm it explicitly:
  "Day X complete. Next session = Day X+1"

TONE RULES:
- End of every topic: ask how they're feeling
  Bored / Good / Tired / Excited
- Adapt accordingly:
  Bored → shorter explanations, more jokes
  Tired → wrap up, give closing quote, stop
  Excited → push harder, give bonus challenge
  Good → maintain current pace

FOR C++ KNOWERS:
- Compare Python to C++ constantly
- Skip absolute basics — they know variables
- Focus only on what Python does differently
- Move fast. They'll keep up.

FOR BEGINNERS:
- Explain everything from scratch
- Real life analogies for everything
- Go slower, more checkpoints
- More praise, more patience
- Never assume they know anything

FOR LAZY/SLOW PACERS:
- Keep sessions short
- One concept per session max
- Don't overwhelm with too much at once
- More jokes to keep energy up

ACTIVELY ASK QUESTIONS:
Not everyone asks questions naturally.
After every explanation, ask:
"Does that make sense or should I break
it down differently?"
"Any why questions before we move?"
Do not wait for them to ask.

# Run this first in terminal:
# pip install reportlab

from reportlab.lib.pagesizes import A4
from reportlab.lib import colors
from reportlab.lib.styles import ParagraphStyle
from reportlab.lib.units import mm
from reportlab.platypus import Paragraph, Spacer, HRFlowable
from reportlab.lib.enums import TA_CENTER
from reportlab.platypus import BaseDocTemplate, PageTemplate, Frame

WIDTH, HEIGHT = A4

def draw_border_on_page(canvas_obj, doc):
    canvas_obj.saveState()
    canvas_obj.setStrokeColor(colors.HexColor("#3776AB"))
    canvas_obj.setLineWidth(6)
    canvas_obj.rect(15*mm, 15*mm, WIDTH - 30*mm, HEIGHT - 30*mm, stroke=1, fill=0)
    canvas_obj.setStrokeColor(colors.HexColor("#FFD43B"))
    canvas_obj.setLineWidth(2)
    canvas_obj.rect(19*mm, 19*mm, WIDTH - 38*mm, HEIGHT - 38*mm, stroke=1, fill=0)
    canvas_obj.restoreState()

def S(size, font='Times-Roman', color="#333333", align=TA_CENTER, space=4, leading=None):
    return ParagraphStyle('s', fontSize=size, fontName=font,
                          textColor=colors.HexColor(color),
                          alignment=align, spaceAfter=space,
                          leading=leading or size*1.4)

def build_certificate(student_name, remarks_text, output_path):
    doc = BaseDocTemplate(output_path, pagesize=A4,
                          rightMargin=32*mm, leftMargin=32*mm,
                          topMargin=28*mm, bottomMargin=22*mm)
    frame = Frame(doc.leftMargin, doc.bottomMargin,
                  doc.width, doc.height, id='normal')
    doc.addPageTemplates([PageTemplate(id='cert', frames=frame,
                          onPage=draw_border_on_page)])
    story = []
    story.append(Paragraph("[ Python 3 ]", S(10, color="#3776AB", space=2)))
    story.append(Spacer(1, 3*mm))
    story.append(HRFlowable(width="80%", thickness=1.5,
                             color=colors.HexColor("#FFD43B")))
    story.append(Spacer(1, 6*mm))
    story.append(Paragraph("Congratulations",
                            S(36, 'Times-Bold', "#3776AB", space=6)))
    story.append(Spacer(1, 5*mm))
    story.append(Paragraph("This is to certify that",
                            S(12, 'Times-Italic', "#666666", space=5)))
    story.append(Spacer(1, 5*mm))
    story.append(HRFlowable(width="35%", thickness=0.8,
                             color=colors.HexColor("#3776AB")))
    story.append(Spacer(1, 3*mm))
    story.append(Paragraph(student_name,
                            S(30, 'Times-Bold', "#3776AB", space=3)))
    story.append(Spacer(1, 3*mm))
    story.append(HRFlowable(width="35%", thickness=0.8,
                             color=colors.HexColor("#3776AB")))
    story.append(Spacer(1, 6*mm))
    story.append(Paragraph(
        "has successfully completed <b>Basics of Python</b>, a structured "
        "introductory course delivered by <b>Con'E</b>, an AI-powered Python "
        "tutor system created by <b>Raj Bhatia</b>.",
        S(11, leading=18, space=4)))
    story.append(Spacer(1, 5*mm))
    story.append(HRFlowable(width="70%", thickness=0.5,
                             color=colors.HexColor("#CCCCCC")))
    story.append(Spacer(1, 4*mm))
    story.append(Paragraph("Topics Covered",
                            S(12, 'Times-Bold', "#3776AB", space=4)))
    for t in [
        "I. Python Boilerplate and Program Structure",
        "II. Data Types and Variables",
        "III. Standard Input and Output",
        "IV. Operators and Mathematical Operations",
        "V. Control Flow Statements",
        "VI. Functions and Code Reusability",
        "VII. Python Standard Libraries",
    ]:
        story.append(Paragraph(t, S(10.5, space=2, leading=15)))
    story.append(Spacer(1, 5*mm))
    story.append(HRFlowable(width="70%", thickness=0.5,
                             color=colors.HexColor("#CCCCCC")))
    story.append(Spacer(1, 4*mm))
    story.append(Paragraph("Remarks",
                            S(12, 'Times-Bold', "#3776AB", space=4)))
    story.append(Paragraph(remarks_text, S(10.5, leading=17, space=4)))
    story.append(Paragraph(
        "This completion demonstrates readiness to explore intermediate "
        "Python topics and begin building basic Python-based projects.",
        S(10.5, leading=17, space=4)))
    story.append(Spacer(1, 5*mm))
    story.append(Paragraph(
        f"Keep learning. Keep hustling, {student_name.split()[0]}.",
        S(13, 'Times-BoldItalic', "#3776AB", space=4)))
    story.append(Spacer(1, 5*mm))
    story.append(HRFlowable(width="80%", thickness=1.5,
                             color=colors.HexColor("#FFD43B")))
    story.append(Spacer(1, 3*mm))
    story.append(Paragraph(
        "Created by Raj Bhatia &nbsp;|&nbsp; "
        "github.com/bakaraj6655 &nbsp;|&nbsp; "
        "linkedin.com/in/raj-bhatia92",
        S(8.5, color="#888888", space=2, leading=13)))
    story.append(Paragraph("Con'E — Python Tutor v1.0",
                            S(8.5, color="#888888", space=2, leading=13)))
    doc.build(story)
    print(f"Certificate saved: {output_path}")
