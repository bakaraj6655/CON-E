# Con'E — AI Python Tutor
> *"Claudey DakAIt ki sikhayi hui Python, Badi KaatilNuma hoti hai."*

Con'E is a portable, document-powered Python tutor that runs entirely through Claude AI. No app. No server. No subscription. Just a document you feed to Claude — and it becomes your personal Python teacher.

Built by a student, for students.

---

## What is Con'E?

Con'E (short for the tutor persona "Claudey DakAIt") is a structured Python learning system packed into a single text file. Feed it to Claude, say `start`, and Claude transforms into a casual, funny, no-nonsense Python tutor that:

- Teaches you Python from absolute scratch
- Adapts to your coding background (beginner or C++ knower)
- Refuses to let you skip the actual coding
- Drops jokes at the right moments
- Gives you a real PDF certificate when you finish

## How To Use

### Step 1 — Download the files
Download these two files from this repo:
- `cone_master_doc_v1.1.txt`
- `cone_master_prompt_v1.1.txt`

### Step 2 — Open Claude
Go to [claude.ai](https://claude.ai) and start a **brand new chat**.

### Step 3 — Upload the master doc
Upload `cone_master_doc_v1.1.txt` to the chat.

### Step 4 — Paste the master prompt
Open `cone_master_prompt_v1.1.txt`, copy its contents, and paste it as your first message. Hit send.

### Step 5 — Say start
Con'E will introduce itself, ask you a few setup questions, and begin teaching.

That's it. No coding required to get started.

## The Certificate

When you finish all 6 days, Con'E generates a personalised PDF certificate with your name, your journey remarks, and credit to the project. You can download and print it.

It is intentionally not professional enough to put on a CV. It is professional enough to feel good about. 😄
